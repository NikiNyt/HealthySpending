# Contributing to HealthySpending

Thank you for your interest in contributing to HealthySpending! We follow a **Meritocracy Model**: contributions are valued based on quality, and ideas have no rank.

### 1. Reporting Bugs
If you find a bug, please create a GitHub Issue using our "Bug Report" template. Include screenshots and steps to reproduce.

### 2. Suggesting Features
Have an idea for the Leaderboard or a new Quest? Open an issue tagged `enhancement`. Describe the feature and how it helps the community.

### 3. Pull Request (PR) Protocol
1.  **Fork** the repository to your own GitHub account.
2.  **Clone** the project to your machine.
3.  Create a **new branch** for your feature:
    `git checkout -b feature/AmazingNewFeature`
4.  Commit your changes with clear messages.
5.  Push to your branch and submit a **Pull Request**.
